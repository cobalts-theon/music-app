Genre card images live here as Android assets.

Use lowercase image filenames with underscores. The current app maps genre names to:
- pop.jpg
- electronic.jpg
- alternative.png
- hip_hop.png
- rock.png
- r_and_b.png
- jazz.jpg
- chill.jpg
- default.png

Replace any file with a new image using the same name to update the matching genre card.
